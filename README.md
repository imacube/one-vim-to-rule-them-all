# Git repos of vim plugins

Create the diretory for the plugins to be stored
```shell
mkdir ~/.vim/pack/git-plugins/start
```

Add the submodules
```
cd ~/.vim/pack/git-plugins/start
git submodule add <URL of git repo to clone>
```
